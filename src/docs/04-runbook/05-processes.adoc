== Prozesse

[role="arc42help"]
****
.Motivation
Eine hohe Verfügbarkeit von IT Services lässt sich nur durch entsprechende Prozesse erreichen. Über diese sollte sich
jedes Teammitglied bewusst sein, daher ist es wichtig, diese zu dokumentieren. Nicht zuletzt ist die
Prozessdokumentation Grundlage für ISO-Audits.

.Inhalt
Die folgenden Prozesse sollten entsprechend den Verfügbarkeitsanforderungen angemessen beschrieben sein:

image::00-processes.service-reliability.drawio.svg[width=70%, scalewidth=12cm]

.Form
Überwiegend Prozessbeschreibung in Form von Tabellen, optional auch grafisch.
